# Instruction

- Android Studio  : Menu Build -> Clean Project -> Run (Shift + F10)
- Terminal
 
 ```
 $cd /path/to/project
 $./gradlew assembleDebug
 ```
- Check out this documentation for more information ([docs](https://documentation.qiscus.com/chat-sdk-android/introduction))
